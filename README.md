**⚠️ 警告：本项目包含NSFW内容**  
本项目收录了部分与角色“流萤”相关的成人漫画。   
本项目仅提供漫画名称，类别，链接等内容，不包含漫画实际内容。   
若你对NSFW内容感到不适，或未达到法定年龄，请立即离开本页面。  

---

## 📖 项目简介

本项目旨在为流萤同好提供一个**离线、可筛选**的NSFW漫画索引工具。  
我收集了一部分流萤漫画，并将其整理为CSV格式。  
当前只收集了禁漫天堂的漫画。  
借助AI开发了一个安卓App(5.0+)，用于读取该CSV文件，支持用户根据类别、标签等条件进行筛选浏览。  
CSV文件已内置在App中，方便离线查阅。
